🌍 Real-World Examples of Stack

🍽️

Stack of Plates

Cafeteria में plates का stack — सबसे ऊपर वाली plate पहले उठाई जाती है। यही LIFO principle है!

🔙

Browser Back Button

जब आप websites browse करते हैं, हर page stack में push होता है। Back button press करने पर top page pop होता है।

↩️

Undo / Ctrl+Z

Text editor में हर action stack में save होता है। Ctrl+Z (Undo) करने पर last action pop होकर reverse होता है।

📚

Stack of Books

किताबों का ढेर — सबसे ऊपर रखी किताब सबसे पहले उठाई जा सकती है, नीचे वाली नहीं।

🎯 Key Concepts / मुख्य बिंदु

01

LIFO Principle

Last In, First Out — जो element सबसे बाद में आता है, वो सबसे पहले निकलता है।

02

TOP Variable

TOP हमेशा stack के सबसे ऊपरी element की position बताता है। Empty stack में TOP = -1 होता है।

03

Overflow Condition

जब stack पूरा भरा हो (TOP = MAX-1) और push करें, तो Stack Overflow होता है।

04

Underflow Condition

जब stack खाली हो (TOP = -1) और pop करें, तो Stack Underflow होता है।
